# build-automation-tool
## Overview

A Bash-based build automation tool that compiles C projects, manages build artifacts, generates logs, and automates application execution.

## Features

- Automatic Compilation
- GCC Integration
- Build Validation
- Build Logging
- Timestamp-based Log Generation
- Automatic Executable Creation

## Technologies Used

- Bash Scripting
- GCC
- Linux
- Build Automation

## How to Run

chmod +x build.sh
./build.sh
